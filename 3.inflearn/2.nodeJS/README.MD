### 1. 노드 시작하기

**Node.js는 크롬 V8 자바스크립트 엔진으로 빌드 된 자바 스크립트 실행기**
서버의 역활도 수행할 수 있다.(**서버가 아님**)

노드는 V8과 libuv를 내부적으로 포함
V8 엔진: 오픈 소스 자바스크립트 엔진] -> 속도 문제 개선
libuv: 노드의 특성인 이벤트 기반, 비동기 I/O 모델을 구현한 라이브러리
